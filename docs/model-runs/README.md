# Historical Review Archive

This directory preserves implementation and review records created under a
retired provenance policy. Existing per-ticket files remain unchanged as
historical evidence. New pull requests do not add agent runtime identity,
reasoning settings, or attribution aggregates; they keep implementation,
`/review`, repair, validation, and guarded-merge evidence in the PR.

The directory name is retained so historical links continue to resolve. For a
new optional durable review record, use `TEMPLATE.md`, which contains only
engineering context, findings, outcomes, and evidence.
